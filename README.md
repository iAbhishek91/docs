# Commands cheatbook

## windows powershell:
> search files and folder:

#### - desc:
  search on the directory path mentioned or in the working directory.
  ##### command:
  ls <path> <foldername> -Directory
  
#### - desc:
  search on all the child path mentiod.
  ##### command:
  ls <path> <foldername> -Directory -Recurse
